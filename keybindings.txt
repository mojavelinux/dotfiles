MBP built-ins:
Delete : actually backspace
Fn+Delete : real delete
Fn+Up : Page Up
Fn+Down : Page Down

Universal:
<Ctrl><Shift>u : enter a unicode sequence

Session:
<Super>l : lock screen
<Super>End : power down options

Applications:
<Super>h : open home directory
<Super>t : launch terminal
<Super>b : open media player

Terminal:
<Ctrl>= : enlarge text
<Ctrl>- : shrink text
<Ctrl>0 : normal text
<Shift>Left : switch to tab to the left
<Shift>Right : switch to tab to the right
<Ctrl><Shift>Left : move tab left
<Ctrl><Shift>Right : move tab right
<Ctrl><Shift>p : paste

Bash:
<Ctrl><Alt>e : expand command alias
<Ctrl>r : reverse incremental history search
<Alt>r : forward incremental history search (broken)
<Ctrl>p : previous in history
<Ctrl>n : next in history
<Ctrl>x<Ctrl>e : edit the commandline
<Alt>s : execute command using sudo

Pidgin:
Ctrl+Up / Ctrl+Down : Paginate through command history (usage: IRC NickServ and ChanServ commands)

Gedit:
<Ctrl>= : enlarge text
<Ctrl>- : shrink text
<Ctrl>0 : normal text

Zoom (in Gnome 3, Zoom must be enabled):
<Ctrl>wheel up : zoom in (doesn't work in Gnome 3)
<Ctrl>wheel down : zoom out (doesn't work in Gnome 3)
<Super>= : zoom in
<Super>- : zoom out

Switcher:
<Super>a : show all windows
<Ctrl><Alt>Down : expose (can drag windows between desktop here)
Menu : show windows on current desktop
<Super>Menu : show all windows
<Ctrl><Alt>Left : switch to desktop to the left
<Ctrl><Alt>Right : switch to desktop to the right
<Alt>Tab : next window (same desktop) (can't use sticky keys)
<Ctrl><Alt>Tab : next window (any desktop) (can't use sticky keys)
<Super>Tab : switch windows in a ring

Touchpad:
<Super><Alt>t : Toggle touchpad (was <Ctrl><Alt>0)
